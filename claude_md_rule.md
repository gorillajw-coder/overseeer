# 글로벌 CLAUDE.md 에 추가할 룰

아래 내용을 `~/.claude/CLAUDE.md` (모든 기기 공통) 에 붙여넣으세요.

---

## 세션 종료 룰 (Overseer)

사용자가 **"/끝"** 이라고 입력하면, 다음 명령을 실행해서 이번 세션을 요약·기록한다:

```bash
python3 ~/.claude/scripts/session_end.py --project-path "$(pwd)"
```

- `--project-path`: 현재 작업 디렉토리. 생략하면 cwd가 자동 사용된다.
- `--session-id`: 생략 가능. 생략하면 해당 프로젝트의 **가장 최근 세션 .jsonl** 을 자동 감지한다.
  (현재 세션 ID를 알고 있으면 `--session-id "<ID>"` 로 명시해도 된다.)

- 세션마다 조용히 기록만 한다(저장+커밋+중앙로그). 알림은 다음날 아침 8시 다이제스트로 나간다.

실행 후 스크립트가 출력하는 결과(`[overseer] ...`)를 사용자에게 간단히 보고한다.

## 깃 동기화 룰 (Overseer)

사용자(주로 Claudy 봇 텔레그램)가 **"깃 동기화해줘"**, **"동기화해줘"** 같은 요청을 하면,
즉흥적으로 git 명령을 판단하지 말고 항상 아래 스크립트를 그대로 실행한다:

```bash
python3 ~/.claude/scripts/git_sync.py
```

- 이 스크립트가 (1) 이 기기의 최근 세션을 지금 바로 체크포인트(요약·커밋·push)하고
  (2) 추적 중인 프로젝트 저장소를 전부 pull한다 — 둘 다 이미 안전하게 구현되어 있으므로
  `git init`, `git add -A`, 새 원격 추가 등을 직접 만들어내지 않는다.
- 실행 후 `[git-sync]` 출력 결과를 사용자에게 간단히 보고한다.
